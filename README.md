## 📌 Overview

This repository contains the source code for my personal portfolio website. Built with a focus on modern web design, it leverages responsive CSS and semantic HTML to tell the story of my career journey—from playing tabletop RPGs in Brazil to building professional systems for the Minecraft Marketplace that reach players globally.

## ✨ Design & Features

- **Story-Driven Structure**: The content flows like an adventure, highlighting projects through a timeline of "chapters."
- **Dynamic Theming**: Intuitive Dark and Light mode toggles using rich CSS properties for smooth, immersive transitions.
- **Responsive Layout**: Fluid typography utilizing `clamp()` and a CSS grid architecture ensure the experience remains stunning across all devices.
- **Zero Dependencies**: Hand-crafted with vanilla HTML, CSS, and minimal JavaScript to ensure maximum performance and maintainability.

## 🛠️ Under the Hood

The codebase reflects my philosophy on building lean, maintainable systems:

### Technologies
- **HTML5**: Leverages semantic tags (`<header>`, `<main>`, `<section>`, `<article>`) for screen readers, accessibility, and SEO.
- **CSS3**: 
  - Extensive use of scoped CSS variables (`:root`) for color palettes, spacing tokens, and typography scales.
  - Complex nested gradients and glassmorphism techniques (`backdrop-filter`) to achieve a premium UI.
- **Vanilla JavaScript**: A lightweight, IIFE-based script that handles theme toggling and accessibility updates without bloating the load time.

### Project Structure
- `index.html`: The core content, inline styles, and logic for immediate rendering.
- `assets/`: Optimized imagery and covers showcasing various Bedrock Add-Ons and maps.
- `CNAME`: Domain configuration for GitHub Pages deployment.
